# 🎙️ VoxOS – Voice-Controlled POS System

A futuristic **voice-operated Point of Sale (POS) web application** that allows restaurant staff to manage orders, billing, and payments **completely hands-free** using voice commands.

---

## 🚀 Project Overview

Traditional billing systems rely on manual input, which can cause delays and errors.
**VoxOS** replaces touch interactions with a **JARVIS-like voice assistant**, enabling users to control the entire system using voice commands.

> Just say **"Hey Vox"** and start managing your business.

---

## ✨ Key Features

### 🎤 Wake Word Activation

* Say **"Hey Vox"** to activate the assistant
* Always-listening system (simulated or ML-based)

---

### 🧠 Voice Command System

Control everything using voice:

* Start new order
* Add / remove items
* Show bill
* Generate invoice
* Select payment method

---

### 🧾 Smart Billing System

* Real-time order tracking
* Automatic total calculation
* Clean bill generation

---

### 💳 Payment System

#### 📱 QR Code Payment (UPI)

* Generates QR code dynamically
* Customers scan and pay instantly

#### 💵 Cash Payment

* Enter amount received
* Auto-calculates change

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/voxos.git
```

2. Open the project folder

3. Run using Live Server

4. The app will automatically open the **Registration Page**
